Building the Inderes Portfolio Dashboard screen-by-screen from the handoff. The user reviews each screen against `design_handoff_portfolio_dashboard/screenshots/NN-*.png` before moving on, and wants to preview periodically.

Decisions (confirmed 2026-06-30):
- **Framework**: Vite + React + TS (no existing app existed — scaffolded fresh at project root).
- **Deployment**: Vercel.
- **AI box**: build stubbed now, wire real Claude (`claude-opus-4-8`) via a `/api/ask` serverless proxy later, grounded with holdings + strategy + targets.
- Charts ported as SVG components rather than a charting library, to match the design 1:1.
- The prototype's `renderVals()` is the data/behavior spec; mock data lives in `src/data/` as the typed "data contract".
